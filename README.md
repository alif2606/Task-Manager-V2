# TaskList V2

A lightweight, transparent desktop task widget for Windows.  
Sits quietly on your wallpaper. No taskbar icon. No distractions.

## Features

| Feature | Status |
|---|---|
| Transparent glassmorphism card | ✅ |
| Always-on-desktop (behind all windows) | ✅ |
| Add tasks with Enter or button click | ✅ |
| Checkbox to mark complete (strikethrough) | ✅ |
| Animated delete (fade out) | ✅ |
| Active tasks above completed tasks | ✅ |
| Drag to move anywhere on screen | ✅ |
| Resize via grip handle | ✅ |
| Auto-save to JSON on every change | ✅ |
| Window position remembered | ✅ |
| Right-click → Clear completed / Quit | ✅ |
| Priority colour dot per task | ✅ |
| Mouse-wheel scrolling | ✅ |
